EMG-Based Effort Classifier

Machine learning · Signal processing · Python

A Python pipeline that ingests multi-channel sEMG signals, extracts time-domain and frequency-domain features, and trains a supervised classifier to distinguish maximal from submaximal voluntary contractions in athletes with physical impairments.

Methods:

  • Band-pass filtering (20–500 Hz) and notch filtering (50 Hz)
  • Root mean square (RMS), mean absolute value (MAV), waveform length
  • Mean power frequency (MNF), median frequency (MDF) via Welch PSD
  • Wavelet packet decomposition for time-frequency features
  • Support Vector Machine (RBF kernel) with nested cross-validation
  • SHAP values for model interpretability
Python scikit-learn SVM Wavelet SHAP

HD-EMG Motor Unit Analysis

Neurophysiology · MATLAB/Python · High-density arrays

High-density EMG decomposition toolbox for extracting individual motor unit action potential trains from 64-channel electrode grid recordings. Used to characterise motor unit properties in Paralympic athletes.

Methods:

  • Convolutive blind source separation (fastICA / gradient CKC)
  • Motor unit discharge rate estimation and inter-spike interval analysis
  • Pulse-to-noise ratio (PNR) quality filtering
  • Cumulative spike train analysis for common synaptic input
  • Synchronisation index between motor unit pairs

Para Classification Evidence Database

Systematic review · Data curation · R

A curated, structured database of peer-reviewed studies on objective measurement tools used in Para sport classification, extracted and synthesised via PRISMA-compliant systematic review methodology.

Methods:

  • Boolean search across PubMed, SPORTDiscus, Web of Science, Scopus
  • Two-reviewer title/abstract and full-text screening (Rayyan)
  • Risk of bias assessment (COSMIN, RoB 2)
  • Narrative synthesis and evidence mapping in R (ggplot2, meta)
Systematic review R Meta-analysis COSMIN

Classification Simulation Protocol

Experimental design · Clinical · Biomechanics

An experimental protocol for healthy volunteers to simulate impairment during standardised functional classification tests, providing ground-truth data for training and evaluating detection algorithms.

Methods:

  • Standardised submaximal effort targets (25%, 50%, 75% MVC)
  • Biofeedback control to maintain target force levels
  • Simultaneous sEMG, force, and motion capture recording
  • Blind randomisation of effort levels across trials
